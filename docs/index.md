# DNS operators

A collection of [Juju](https://canonical.com/juju) {ref}`charms <juju:charm>` that deploy a fully functional [DNS](https://en.wikipedia.org/wiki/Domain_Name_System) solution, facilitating the integration of DNS services with other charms.

The goal is to provide an easy-to-use and reliable DNS deployment that offers a simple integration for your existing charmed applications, enabling them to request resource records and automate DNS processes.
